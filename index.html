What is css?

  1. CSS stands for Cascading Style Sheet.
  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.
  3. CSS is not neccesary CSS saves a lot of work. can control the layout of multiple web pages all at once<
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML document in 3ways:

  • Inline-by using the style attribute inside HTML elements1
  • Internal-by using a style element in the head section
  • External-by using a link element to link to an external CSS file.
  • The most common way to add CSS,is to keep the styles in external CSS files.However, this tutorial we will use inline and internal styles, this is easier to demonstrate,and easier for you to try it yourself.

when the comparing CSS class vs ID,the difference is that class applies a style to multiple elements.ID,on the other hand,applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and its used by javascript.
